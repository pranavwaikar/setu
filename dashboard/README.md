# Setu Dashboard Service

This is the Next.js frontend dashboard for the Setu Tunnel Platform.

## Running the Service

Setu is a containerized, Docker-first platform. To run this service alongside the rest of the Setu stack (PostgreSQL, API, and Go Gateway), please refer to the main [README.md](file:///Users/pranavwaikar/Documents/GitHub/setu/README.md) in the project root.

You can start the entire stack with:

```bash
docker compose up --build -d
```
