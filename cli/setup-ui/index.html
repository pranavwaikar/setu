1. Host Header Overrides

Rewrite Host header to match destination target (for Vite, Next.js, Webpack):

setu expose 3000 --subdomain my-app --host-header rewrite

2. Local HTTPS Endpoints

Proxy to local HTTPS servers, skipping self-signed SSL certificate check:

setu expose https://localhost:3000 --subdomain my-app --insecure-skip-verify

3. Edge Protection (Basic Auth)

Require credentials before public visitors can access your tunnel:

setu expose 3000 --subdomain my-app --auth "admin:secret"

4. Layer-4 TCP Tunneling

Tunnel raw TCP protocols (PostgreSQL, MySQL, SSH) via dynamic high-ports:

setu expose 5432 --subdomain my-db --tcp

5. Persistent Local Inspection UI

Visually inspect incoming HTTP headers, payloads, and response status codes at: http://localhost:4500/inspect
